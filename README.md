# 🪐​ Cowboy BebOOP

RPG narrativo a scelte multiple ambientato in un futuro distopico ispirato a *Cowboy Bebop*.  
Il giocatore veste i panni del capitano della **Blue Mantis**, una nave da bounty hunting coinvolta in una cospirazione galattica. Tre episodi, otto finali, nessuna scelta innocua.

---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25
- Gradle

### Istruzioni

```bash
git clone https://github.com/<utente>/it.unicam.mpgc.rpg126421.git
cd it.unicam.mpgc.rpg126421
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```
